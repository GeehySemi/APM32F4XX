3. 请确认器件型号、开发板、存储器配置及工具链与实际硬件一致。

## English

This repository contains reference documents and example resources for Geehy APM32F4 series microcontrollers.

### Repository contents

| File | Description |
| --- | --- |
| `APM32F4xxx用户手册 V2.0.pdf` | APM32F4 series user manual (Chinese) |
| `APM32F407VGT6S数据手册 draft V0.4.pdf` | Draft APM32F407VGT6S datasheet (Chinese) |
| `APM32F4xx_SDRAM.zip` | SDRAM reference package for the APM32F4 series |

### Usage

1. Download the document or archive required for your project.
2. For the SDRAM package, extract the archive and review the included project configuration before building.
3. Confirm that the selected device, board, memory configuration, and toolchain match your hardware.

### Important notice

Some files in this repository are historical or draft materials. Always refer to the [Geehy website](https://www.geehy.com/) for the latest product documentation, software, and device information before starting a production design.

### Support

For product information and technical support, visit [www.geehy.com](https://www.geehy.com/).
